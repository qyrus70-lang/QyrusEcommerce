# QYRUS ECOMMERCE

----
Demo Ecommerce application which is a ecommerce sandbox without integration with payment gateway. This is a mono repo consisting of python backend and react frontend.

## Features:
✅ Signup, Login and Forgot Password
✅ Products, Product Details
✅ Search, Categories
✅ Add to Cart, Instant Buy
✅ Wishlist
✅ Manage Orders
✅ Compare Products
✅ Checkout and Payments Page
✅ Manage Addresses
✅ Review and Ratings

## Local Setup:
Frontend:
clone the repo and go the directory
```
> cd QyrusEcommerceFrontend
> npm i
> npm run dev
```

Backend:
```
cd QyrusEcommerceBackend
python app.py
```

## Test:
Open the browser with the url http://localhost:5173
